Creating project node
yarn init -y

Include the express  lib and types
yarn add express

yarn add @types/express -D

Create folder src and file service.ts

install dependency typescript

yarn add typescript -D

yarn tsc --init

descomment outDir

execute yarn tsc

yarn eslint --init
